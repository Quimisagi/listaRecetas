# RecetasCocina

```
Instalar mux router:
go get -u github.com/gorilla/mux
```

```
go build
./recetasDeCocina
```


## Endpoints

### Ver todas las recetas

```
GET api/listaRecetas
```
### Ver una receta

```
GET api/listaRecetas/{id}
```
### Agregar una receta

```
GET api/listaRecetas
```

### Actualizar una receta

```
GET api/listaRecetas/{id}
```

### Eliminar una receta

```
GET api/listaRecetas/{id}
```
